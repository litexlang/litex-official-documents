## 1. Core Updates

1. Chained equality now works: `1 + 1 = 2 = 2 * 1` is now a valid statement
2. Improved output messages: Both errors and successes now display line numbers, making it easier to locate code issues and helping AI models learn more from the output
3. Fixed several bugs, improving Litex performance by up to 1000% in certain scenarios

## 2. AI-Related Updates

1. litex-gsm8k-killer completed the GSM8K dataset with 90% single-pass accuracy and open-sourced the dataset on Hugging Face
2. litex-task-verifier provides syntax and semantic checking for Litex with 99.11% accuracy
3. Released Litex for Math23K and Litex for Math-500 datasets on the official website (pending resolution)
4. A community contributor provided an LLM prompt for translating Lean's mathlib to Litex, showing promising results
